#Instalando o Linux Slitaz em uma Virtual Box

* Instalando a Virtual Box
	* 1. Acesse o site: "https://www.virtualbox.org/".
	* 2. Escolha a versão compativel com o sistema operacional hospedeiro e inicie o download.
	* 3. Abra o arquivo executavel.
	* 4. Siga o procedimento padrão de instação clicando nos botões "Next" e "Yes", e por fim clique em "Finish".
